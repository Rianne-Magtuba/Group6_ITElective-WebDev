Web Development

From: Web Development.pdf

World Wide Web (WWW)

is a system of internet service that support formatted documents such as HTML, HXTML, XML, and PHP.

Tim Berners-Lee

a British scientist, invented the World Wide Web (WWW) in 1989

Internet

a global interconnection of networks.

Web Browser

a software application used to locate, retrieve and display content on the World Wide Web, including Web pages, images, video and other files.

Types of Browsers

Camino, Chrome, Edge, Firefox, Safari

IP

Internet Protocol

TCP

Transport Control Protocol

Protocol

is a set of rules specifying how computers should communicate with each other over a network.

Packet

Data sent across the Internet is called a message.

Packet Routing Network

It is a network that routes packets from a source computer to a destination computer.

Web Server

sends the information back to the Web browser which displays the results on the computer or other Internet-enabled device that supports a browser.

Hypertext Transfer Protocol (HTTP)

allows for the retrieval of linked resources from across the Web.

Hypertext Transfer Protocol Secure (HTTPS)

provides authentication of the visited website and protects the privacy and integrity of exchanged data.

Website

is the combination of webpages.

Webpage

is a single page composed of different web components like image, video, information, etc.

Uniform Resource Locator (URL)

it is the global address of documents and other resources on the World Wide Web.

Static Website

is the simplest kind of website; it contains information that does not change. Most static websites are created for viewing purposes only.

Dynamic Website

has more functionality, flexibility and features.

Different Types of Websites
  • Personal Website - is primarily used for informative or entertainment purposes but can also be used for personal career marketing, social networking, or personal expression.
  • Company Website - a necessity for entrepreneurs, small businesses, home-based businesses, and anybody selling products or services.
  • E-Commerce Website - provides shoppers with the convenience of a fully integrated free shopping cart and a secure checkout experience directly from the website.
  • Government Website - is built to provide interactions and services between the citizen, employee, government, and government agencies.
  • Social Networking Website - enables users to create profile within the website among people who share interests, activities, photos, videos, backgrounds or real-life connections.
  • Entertainment Website - is one of the most used and viewed websites in the world.
  • Blogging Website - contains and maintains a blogger's or group of bloggers' own experiences, observations, suggestions, opinions, etc.
  • Search Engine and Portal Website - a website specially designed to provide a starting point for web consumers, and brings information together from diverse sources in a uniform way.
HTML Tags

HTML (HyperText Markup Language) - is the most basic building block of the Web. It defines the meaning and structure of web content. First created by Tim Berners-Lee, Robert Cailliau, and others starting in 1989.

"Hypertext" - refers to links that connect web pages to one another.

Anatomy of an HTML element:

  • Opening tag - consists of the element name in angle brackets.
  • Closing tag - same as opening tag but with a slash before the element name.
  • Content - the content of the element.
  • Attributes - extra information about the element.
  • Nesting - elements can be placed inside other elements.
  • Empty elements - elements with no content.
Anatomy of an HTML Document
  • <!DOCTYPE html> - required preamble.
  • <html></html> - root element.
  • <head></head> - container for meta information.
  • <meta charset="utf-8"> - sets character set.
  • <title></title> - page title in browser tab.
  • <body></body> - visible page content.

CSS

Cascading Style Sheets (CSS) - a design language to simplify styling web pages.

  • Advantages: saves time, faster loading, easy maintenance, superior styling, multi-device compatibility, global standards.
  • Types: Inline CSS, Internal CSS, External CSS.
Inline CSS Example
<p style="color:#009900; font-size:50px; font-style:italic; text-align:center;">
GeeksForGeeks
</p>
Internal CSS Example
<style>
.main { text-align:center; }
.GFG { color:#009900; font-size:50px; font-weight:bold; }
.geeks { font-style:bold; font-size:20px; }
</style>
External CSS Example
<link rel="stylesheet" href="style.css">
CSS Syntax
  • Selector - the HTML tag to style.
  • Property - an attribute like color, border, etc.
  • Value - assigned to properties, e.g., red or #F1F1F1.
CSS Selectors
Fonts and Text
  • font-family, font-style, font-variant, font-weight, font-size, font shorthand.
CSS Boxes
  • Padding - space around content
  • Border - line around padding
  • Margin - space outside the border
CSS Links
  • :link - unvisited hyperlinks
  • :visited - visited hyperlinks
  • :hover - when pointer hovers
  • :active - element being clicked
CSS Tables
  • border-collapse, border-spacing, caption-side, empty-cells, table-layout
